## Code of Conduct

This project follows the Contributor Covenant code of conduct. By participating, you agree to:

- Be respectful and considerate in all communications.
- Use welcoming and inclusive language.
- Show empathy and patience toward other contributors.

Unacceptable behaviour includes harassment, hateful language, personal attacks, and other actions that make contributors uncomfortable. Maintainers will review reports and may remove or restrict contributors who violate this policy. This Code of Conduct is based on the widely-used Contributor Covenant; see https://www.contributor-covenant.org/ for more detail.
